# Floating Point ALU using IEEE-754 (Verilog HDL)

## Project Overview
This project implements a **Floating Point Arithmetic Logic Unit (FP-ALU)** using **Verilog HDL** following the **IEEE-754 single precision standard**. The design supports basic floating-point arithmetic operations and is developed and verified using AMD Vivado.

The project demonstrates the complete FPGA design flow including design entry, simulation, synthesis, timing analysis, and implementation.

---

## Features
- IEEE-754 Single Precision (32-bit)
- Modular Verilog Design
- Behavioral Simulation
- Synthesis and Implementation
- Static Timing Analysis
- Post-Synthesis Simulation
- Post-Implementation Simulation

---

| Operation | Description |

| Multiplication | Floating point multiplication |
| Division | Floating point division |
